Руководство пользователя
Введение
Данное приложение предназначено для создания и тестирования симуляций четырехмерных фигур. В этом руководстве вы найдете подробные инструкции по использованию интерфейса приложения, а также ответы на часто задаваемые вопросы.Запуск приложения
Окно авторизации

При запуске приложения откроется окно авторизации. Вам необходимо ввести имя пользователя (логин) и пароль.
Если у вас нет учетной записи или вы забыли данные для входа, нажмите кнопку «Зарегистрироваться».
После нажатия на кнопку будет создан новый аккаунт, а также личный репозиторий пользователя.


Работа с симуляцией
Ввод конфигурации симуляции

После успешной авторизации вы попадете на окно «Ввод конфигурации симуляции».
В этом окне вы можете добавлять и удалять четырехмерные фигуры, задавать их начальные координаты, размеры и цвета.
Чтобы сохранить параметры фигуры в список для запуска симуляции, нажмите кнопку «Добавить фигуру (+)».



Пример добавления фигуры

Например, вы можете добавить тессеракт с начальными координатами в начале координат и цветом #ff4bed.
Чтобы сохранить параметры симуляции, нажмите кнопку «Сохранить пресет». Откроется окно, где вам нужно ввести имя сохраняемого файла.



Загрузка пресета

Чтобы загрузить ранее сохраненный файл, нажмите кнопку «Загрузить пресет». Откроется окно со списком файлов из вашего личного репозитория.


Помощь и поддержка
Руководство пользователя

Если у вас возникли вопросы по функционалу приложения, нажмите кнопку «Руководство пользователя». Откроется файл Readme.txt, в котором описан процесс создания и тестирования симуляции, а также приведены ответы на часто задаваемые вопросы.


Запуск симуляции
Запуск симуляции

Для запуска симуляции нажмите кнопку «Запуск симуляции». Обратите внимание, что в списке должна быть добавлена хотя бы одна фигура.
После запуска откроется окно симуляции, в верхнем левом углу которого будут отображены координаты центра фигуры.


Управление фигурами
Управление фигурами

Пользователь может поворачивать фигуры в 3D-пространстве, перемещая мышь.
Для поворота фигур в 4D-пространстве используйте клавиши:

↑ - вверх
↓ - вниз
← - влево
→ - вправо


Чтобы запустить движение фигуры, нажмите кнопку «Tab».
Если произойдет столкновение фигур, движение остановится, и вы сможете осмотреть момент коллизии со всех ракурсов.


Заключение
Это руководство предоставляет основные инструкции по использованию приложения для симуляции четырехмерных фигур. В случае возникновения дополнительных вопросов, пожалуйста, обратитесь к разделу «Руководство пользователя» или свяжитесь с поддержкой.
